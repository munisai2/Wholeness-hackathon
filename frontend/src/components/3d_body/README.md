# 3D Body Model Feature

This directory contains the components for the interactive 3D human body model.

## key Components
- `BodyModel.tsx`: The Three.js/React-Three-Fiber canvas component.
- `InteractionLayer.tsx`: Handling clicks and rotations on body parts.
- `BodyPartOverlay.tsx`: The UI overlay when a part is selected.

## Libraries
- three
- @react-three/fiber
- @react-three/drei
